Curriculum vitae: Joris Putteneers

// Personal Profile:
My name is Joris Putteneers, I consider myself a speculative architect, blurring the boundaries between fiction and nonfiction architecture. Mapping the post-Anthropocene through a nonhuman ontology.
I’m currently a senior teaching fellow at the Bartlett School of Architecture. I’m also actively working in my own practice since 2017, which specializes in:
· Data driven & Computational design · VR/AR technology and implementation · Digital fabrication · Concept design & Art direction · Visualization

Description

// I use SideFX Houdini for procedural work, or for the generation of synthetic datasets. Houdini has been my main driver for over 7 years. When working with geometric models, I mainly output renders from the Karma renderengine. My Houdini expertise extends SOP, VOP, LOP, PDG, CHOP & COP levels. I use vex (based on C++) & Hscript as a way to optimize node structures and to express computational Logic inside the Houdini Framework.
// I use Blender (+ Cycles, Eevee) for visualisation, world building, art direction and prototyping. It is a fast, intuitive and versatile mesh based modeling software that allows me to rapidly sketch possible outcomes.
// I use Python as a way to automate tedious tasks, mainly inside Houdini. I also use Python for machine learning purposes.
// For optimizing my logistical processes, I use Git as my Version Control platform.
// I mainly use Linux (ubuntu distribution) as my OS. This prevents high-performance bottlenecks
// All other tools I use on a weekly or monthly basis, depending on the current project.
